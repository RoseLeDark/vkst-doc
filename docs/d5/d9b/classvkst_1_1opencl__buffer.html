Ausführliche Beschreibung

template<typename TBUFFER, cl_mem_flags TFLAGS = CL_MEM_READ_WRITE>
class vkst::opencl_buffer< TBUFFER, TFLAGS >

A template class representing an OpenCL buffer with customizable flags.

Template-Parameter
TBUFFERThe type of the buffer (e.g., a custom buffer type).
TFLAGSThe OpenCL memory flags to be used for the buffer (default: CL_MEM_READ_WRITE).

This class provides functionality to create and manage an OpenCL buffer, including operations such as transferring data to/from the buffer and supporting compression.
