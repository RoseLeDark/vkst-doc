Ausführliche Beschreibung

template<typename TBUFFER>
struct vkst::lz4_compression_filter< TBUFFER >

LZ4-based compression filter.

This filter performs compression and decompression using the LZ4 algorithm. It is enabled only if the NO_LZ4_COMPRESSION macro is not defined.

Template-Parameter
TBUFFERType of buffer to compress/decompress.
