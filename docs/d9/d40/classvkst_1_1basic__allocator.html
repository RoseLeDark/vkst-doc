Ausführliche Beschreibung

template<class TAllocator, class TFilter = basic_allocator_filter>
class vkst::basic_allocator< TAllocator, TFilter >

The basic allocator for all allocator implementations in this library.

This class provides a basic allocator that combines an allocator implementation (passed as TAllocator) with a filter (passed as TFilter). The filter is used to perform actions before and after allocations and deallocations. It provides methods to allocate and deallocate memory, as well as construct and destroy objects.

Template-Parameter
TAllocatorThe allocator implementation type.
TFilterThe filter type used for pre- and post-allocation operations.
