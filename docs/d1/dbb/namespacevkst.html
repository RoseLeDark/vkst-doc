◆ default_limmeted_allocator

template<size_t TMaxAlloc>
using vkst::default_limmeted_allocator = typedef malloc_allocator<basic_allocator_maximal_filter<TMaxAlloc> >

A default allocator with a maximum allocation size limit.

This template alias defines a default allocator using the malloc_allocator template, but with a memory filter that enforces a maximum allocation limit (TMaxAlloc). The allocator will restrict allocations to the specified size, preventing the total memory usage from exceeding the limit during the program's runtime.

Template-Parameter
TMaxAllocThe maximum memory size (in bytes) that the allocator can allocate.

◆ malloc_allocator

A wrapper for the basic_malloc_allocator with a filter for allocation events.

This alias provides an easy-to-use wrapper that associates a memory filter with the basic_malloc_allocator. The filter can be used to log or modify allocation/deallocation behavior, or impose limits on allocation sizes.

Template-Parameter

◆ logging_level

enum class vkst::logging_level
strong

Enum representing different logging levels.

The logging levels control the verbosity and type of messages logged.

  • Debug: Used for detailed information, typically useful for debugging.
  • Info: General informational messages.
  • Warn: Indicates a potential issue that doesn't interrupt execution.
  • Error: Represents an error that affects functionality but allows continued operation.
  • Critical: Critical errors that likely lead to application termination.
